Form Submission Flow Test

This page tests the complete form submission flow including validation, webhook integration, and feedback.

Test Instructions:

  1. Fill out the form with valid data
  2. Check the KVKK consent checkbox (submit button should enable)
  3. Click "Gönder" button
  4. Observe the loading indicator (spinner + "Gönderiliyor..." text)
  5. Check for success/error message at the top of the form
  6. Verify form resets after successful submission

Note: The webhook endpoint may not be available, which will result in an error message. This is expected for testing purposes.
